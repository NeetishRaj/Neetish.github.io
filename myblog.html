Nov 23, 2017

How is my Icon for this website, that you can see in the tab at the top of this window. Should have done this earlier but thought that It would be too tough but turns out making Favicons is so easy you just need a 16x16 or 32x32 size image file saved with .ico extension and save it to your main server directory in the where your index file is.

But there are complications involved if you want it to be perfect, because you ll need various formats of your icon for various platforms i.e apple, windows, android etc. They have their own styling for icons, so we ll need to display platform specific images. Also we need to have an SVG format too, in case the image need to scale without pixelleting and also a small memory-size alternative for your icon Image as Fallback in case main one does'nt load due to high bandwidth.

You can check out my favicon folder here. Although you dont need to do this because there might be only slight difference mainly in background fill color, margin, padding, border-radius or any image filters. But if wanna preserve the actual image quality in all platforms then it is mandatory to have all backups. If you want your own cross platform favicon sets then its very easy if you use any online generators like this Favicon Generator that comes very handy and Free of cost also can tweak it to your desires platform-wise.
