List of Projects

Driver assist system

A portable and practical Networking modules to assist drivers in a RSU range using RF chips, Arduino, sensors etc. It monitors and alerts during presence of other vehicles in Low-visibility scenarios and can detect obstacles with Priority Allocation and Rash Driver Profiling features. This project augments the capabilities of a self-driving vehicle.
Contribution: Hardware Configuration and source code for all the modules
Technology used: Wireless Sensor Networks, C++, X-CTU, MANETs, V2X, IEEE standards, Arduino, Xbee

Location privacy in Geo-soacial appls

It masks the Geo-location data for Geo-Social apps in AndroidOS and allows registered users to choose among preset of location for the Geo-Social app. Contribution:Front end and Google map API Technology used:Eclipse, Android SDK, Google Map API, Java, AndroidOS
