Introduction

  1. This webpage serves as a portal to my online presence and all relevant informations about my works and contributions are just a click away.
  2. This website is divided into 5 parts (although few pages still need work)
    • Home: All important stuffs are hyper-linked here

    • Projects: Details and links of all projects old or ongoing will be present here
    • My Blog: A place to vent out my stress and hoping that somebody will read it.
    • Contact Me:: Just a typical page one would find in websites, name suggests the purpose

Development Note

  1. It all sarted with a plain HTML page with embedded style and scripts.
  2. Then the index page started getting complex and styles were clashing, so grouped all the style details in a single style tag in the header section
    3. Later on when more html pages were added, then it made complete sense to make a separate style.css file to handle all HTML pages.
  3. Although not a lot of script was used in the beginning as I thought that it would not be supported on Github static pages but to my amazement you can add variety of script features including animation, event-handling, etc although backend support will be tough here better deploy on a paid server
  4. So made a separate JS directory to handle and store all JavaScript codes.
  5. Then jQuery (JavaScript Library) came to my life and it was a revelation, JS codes became so much shorter and much more functional that I chose to replace all JS codes in jQuery
  6. Also DOM(Document Object Model) concept was more clear and JSON(Javascript Object Notation) started to make sense.
  7. Moreover, using jQuery was I able to understand the working of AJAX
  8. Right know I am exploring possibilities with electron
    9. a cool open source initiative for development of Cross-platform applications.
