Note the side effects of float, it has eaten the bottom padding applied to all nav <divs>, to see it press the given button to remove this para and see the difference in bottom padding again on this nav-bar and compare from upper nav-bars.           display:flex;

          border-bottom: 5px solid grey;
          border-top: 5px solid grey;
          padding: 0;
          list-style: none;
          background-color: #e8e8e9;
        }
        li {
          background-color: #cacaca;
          margin: 8px;
          padding: 4px;
          border-radius:20px;
          text-decoration:none;
          flex:0 1 auto;
          transition: 0.5s;/*background-color 0.6s, color ease-in-out 0.8s, transform 0.6s;
          transition-timing-function: cubic-bezier(10,5,2,1);*/
         }
         li:hover{
           background-color:#4a4a4a;
         }
         li a:hover{
           color:#cacaca;
           list-style-type: none;
         }
         li:nth-child(odd):hover{
             transform:rotate(7deg);
         }

         li:nth-child(even):hover{
             transform:rotate(-7deg);
         }
         li:nth-child(1){
           margin-right:auto;
