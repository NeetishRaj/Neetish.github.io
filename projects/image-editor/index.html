Designing webpages is easy
Designing beautiful webpages is hard
But designing responsive layouts is Headache

To make your beautiful layout to hold good for every devices is a nightmare, offcourse there are Bootstrap classes available for that, but what I felt in my learning was that a good designer must be able to create his own Bootstrap equivalent classes.

This project has nothing fancy but hard work and detailed designing, its still not that good looking and requires further refinement in:-

  • color combinations
  • Layout alignments
  • Icon selections
  • Transitions
  • Ease of use
  • Lesser dependence on script
