Since from recent years, images are being more prominent in the front facade of web designs, hence its logical to understand all the complexities involved while handling images and to know few tactics too. This web-app is still undergoing improvements and the recent version can be checked by clicking on the image.

Color Picker Tool

While designing layouts its inevitable to get stuck in picking color combinations for your layout and time is of great essence. There are many good tools available, my favourite being Adobe Color wheel. But I needed a light weight version for my use, moreover it could be used as a study tool for beginners as well. Its based on simple HTML and CSS with javascript to handle input events and results. I think it has a lot of potential to grow more like a color diary for designers.
